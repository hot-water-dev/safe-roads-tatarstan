## Описание проекта

Интерактивная система анализа и визуализации аварийности на дорогах Республики Татарстан. 
Цель — выявить очаги концентрации ДТП со смертельным исходом и сформировать **ИИ-рекомендации** по снижению аварийности для каждого очага.

Система автоматически находит опасные участки на карте, анализирует их статистику и через нейросеть **GigaChat** выдаёт конкретные меры: где поставить светофор, где улучшить освещение, где ограничить скорость.

## Данные

**Источник:** Официальный портал [stat.gibdd.ru](https://stat.gibdd.ru/)  
**Тип данных:** Выгрузки в формате **XML** за 2025–2026 годы по Республике Татарстан  
**Объём:** 4 366 ДТП, 7 559 транспортных средств, 10 910 участников

## Используемые технологии

- **Backend:** Python — FastAPI, Pandas, Scikit-learn (DBSCAN), SQLite
- **Frontend:** TypeScript — React, Leaflet (react-leaflet), Vite
- **AI:** GigaChat API (Сбер) — бесплатный российский LLM-сервис
- **Карты:** OpenStreetMap через Leaflet
- **Панорамы:** Яндекс.Панорамы
